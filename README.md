**Guftagu** is a modern, full-stack real-time chat application built with production-ready technologies. It enables seamless communication through one-to-one messaging, group chats, user authentication, image sharing, and live user presence tracking. The application is designed for performance, scalability, and a clean user experience using modern web technologies.

---

## Features

#### Real-Time Messaging

* Instant one-to-one chat
* Real-time message delivery and updates

#### Group Conversations

* Create and manage group chats
* Multi-user communication with dynamic group management

#### User Profiles

* Update name and profile image
* Personalized user experience

#### Live Online Status

* Track active users in real-time

#### Image Sharing

* Upload and share images within chats via Cloudinary

#### Multiple Authentication Methods

* Sign in or sign up using email, GitHub, Google, or Twitter

## Tech Stack

* **Next.js**: React framework for server-side rendering and static site generation
* **TypeScript**: Strongly-typed JavaScript for safer and scalable code
* **Tailwind CSS**: Utility-first CSS framework for responsive UI design
* **Headless UI**: Accessible and unstyled React components
* **React**: JavaScript library for building user interfaces
* **Prisma**: Modern ORM for database modeling and querying
* **MongoDB**: NoSQL database for storing chat data
* **NextAuth**: Authentication framework for Next.js applications
* **Zustand**: Lightweight state management library
* **Pusher**: Real-time messaging and presence detection
* **Cloudinary**: Cloud-based image management and hosting
